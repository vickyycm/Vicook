# 🍳 Vicook – Plataforma Web de Recetas

Este proyecto fue desarrollado como parte de mi formación en la materia de Análisis y Metodología de Sistemas, con el objetivo de crear una aplicación web dinámica orientada a la gestión y visualización de recetas de cocina.

La propuesta consiste en una plataforma donde los usuarios pueden explorar recetas, interactuar con el contenido y gestionar información relacionada con preparaciones culinarias, todo dentro de una interfaz clara y organizada.

---

## ✨ Características

- Sitio dinámico desarrollado con Flask (Python).
- Organización de recetas por categorías (ej: ingredientes, preparación).
- Interfaz web simple, clara y enfocada en la experiencia del usuario.
- Separación de lógica y estructura (HTML, CSS y backend).
- Proyecto pensado para ejecutarse en entorno local.

---

## 🧠 Tecnologías utilizadas

- Python (Flask)
- HTML5
- CSS3
- (Opcional según avances: JavaScript)
